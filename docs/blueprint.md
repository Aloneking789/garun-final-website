# **App Name**: Garun App Platform

## Core Features:

- Hero Section: Display a captivating hero section with a headline, subtext explaining free consultancy, WhatsApp and Instagram CTA buttons, and a phone mockup with floating animation.
- Service Vertical Sections: Showcase six service verticals with alternating text and image layouts, including headings, subheadings, descriptions, feature lists with tick icons, and a WhatsApp CTA button for each vertical.
- Application UI Gallery: Implement a horizontal carousel/slider to display multiple phone mockups showing various app screens for different platforms, with slider dots/arrows and a smooth sliding animation.
- Testimonials Section: Feature a testimonial slider with soft card designs and subtle slide/fade animations, displaying quotes, client names, and avatars for user testimonials.
- Brand/Clients Strip: Include a horizontal row of client/brand logos that transition from grayscale to color on hover, showcasing partnerships and collaborations.
- SAAS Plan Pricing Section: Highlight a pricing card called 'Startup Plan' with an icon, bullet points detailing the included features (Android & iOS apps, admin panel, support), pricing, and a WhatsApp CTA button.
- Final CTA and Footer: Include a concluding section with a CTA, tagline, WhatsApp and Instagram buttons, a phone mockup image, copyright information, and a 'back to top' button icon.

## Style Guidelines:

- Primary color: Light cyan (#A0D9D9) for a fresh, tech-forward feel.
- Background color: Very light cyan (#E0F8F8).
- Accent color: Light green (#A0D995) to give the design a little more punch.
- Body and headline font: 'Inter', a sans-serif font, will be used to maintain a neutral, clean, and modern look.

- Clean, modern SaaS landing page style with lots of whitespace, rounded cards, and subtle shadows. Fully responsive design for mobile, tablet, and desktop.
- Smooth scroll to sections. Hover scale and shadow on cards and buttons. Subtle parallax or float on main phone mockups. Light animations on scroll and hover.
